Из двух старших классов, насчитывающих по 303030 учеников каждый, надо выделить по одному ученику для помощи в уборке столовой. Сколькими способами может быть сделан этот выбор?

Решение

Ученика из первого из двух классов можно выбрать 303030 разными способами. Вне зависимости от сделанного выбора, второго ученика из второго класса тоже можно выбрать 303030-ю способами. По правилу умножения получаем всего 30⋅30=90030 \cdot 30 = 90030⋅30=900 способов выбрать пару учеников для уборки в столовой.

Ответ

900900900

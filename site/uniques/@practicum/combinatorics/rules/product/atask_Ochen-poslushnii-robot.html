Очень послушный робот
ЕГЭ Информатика

Артем создал робота, который умеет делать три арифметических действия:

  1. Прибавить 111

  2. Прибавить 222

  3. Умножить на 222

Сколько различных цепочек действий можно дать роботу, чтобы он из числа 333 получил число 121212 и чтобы в промежуточных вычислениях обязательно оказалось число 888?

Указание

Задание решается практически точно так же, как и аналогичная задача в теме «Правило суммы».

Решение

Обозначим за RnR_nRn​ количество цепочек действий, которые дают в результате число nnn. Формулы для RnR_nRn​ получатся почти такими же, что и в уже разобранной аналогичной задаче.

Rn=Rn−1+Rn−2(n нечетное)Rn=Rn−1+Rn−2+Rn2(n четное) R_n = R_{n-1} + R_{n-2} \quad (n \ \text{нечетное}) \\ R_n = R_{n-1} + R_{n-2} + R_{\frac{n}{2}} \quad (n \ \text{четное})Rn​=Rn−1​+Rn−2​(n нечетное)Rn​=Rn−1​+Rn−2​+R2n​​(n четное)

Получить число 333 можно единственным способом — если не дать роботу никаких команд. Поэтому R3=1R_3 = 1R3​=1. Получить 444 можно только одним способом — единожды применив действие (1)(1)(1). Поэтому R4=1R_4 = 1R4​=1.

В условии задачи, говорится, что робот в процессе выполнения действий обязательно должен получить число 888. Поэтому сначала ищем R8R_8R8​.

R5=R4+R3=1+1=2R6=R5+R4+R3=2+1+1=4R7=R6+R5=4+2=6R8=R7+R6+R4=6+4+1=11 R_5 = R_4 + R_3 = 1 + 1 = 2 \\ R_6 = R_5 + R_4 + R_3 = 2 + 1 + 1 = 4 \\ R_7 = R_6 + R_5 = 4 + 2 = 6 \\ R_8 = R_7 + R_6 + R_4 = 6 + 4 + 1 = 11R5​=R4​+R3​=1+1=2R6​=R5​+R4​+R3​=2+1+1=4R7​=R6​+R5​=4+2=6R8​=R7​+R6​+R4​=6+4+1=11

Введем теперь обозначение PnP_nPn​, которое показывает количество цепочек действий для попадания из числа 888 в число nnn. P8=P9=1P_8 = P_9 = 1P8​=P9​=1 по том же причинам, по которым R3=R4=1R_3 = R_4 = 1R3​=R4​=1. Действие (3)(3)(3) мы уже не можем использовать, иначе перелетим за 121212, поэтому Pn=Pn−1+Pn−2P_n = P_{n-1} + P_{n-2}Pn​=Pn−1​+Pn−2​.

P10=P9+P8=1+1=2P11=P10+P9=2+1=3P12=P11+P10=3+2=5 P_{10} = P_9 + P_8 = 1 + 1 = 2 \\ P_{11} = P_{10} + P_9 = 2 + 1 = 3 \\ P_{12} = P_{11} + P_{10} = 3 + 2 = 5P10​=P9​+P8​=1+1=2P11​=P10​+P9​=2+1=3P12​=P11​+P10​=3+2=5

Итак, у нас есть 111111 способов добраться до числа 888, и при любом выбранном способе затем есть 555 вариантов добраться до числа 121212. По правилу умножения получаем 11⋅5=5511 \cdot 5 = 5511⋅5=55 возможных цепочек действий.

Ответ

555555

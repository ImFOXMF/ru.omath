На рисунке — схема дорог, связывающих города. По каждой дороге можно двигаться только в одном направлении, указанном стрелкой. Сколько существует различных путей из города AAA в город KKK?

Указание

Принцип решения точно такой же, как и рассмотренного в теории примера.

Решение

Общие рассуждения и пояснения насчет хода решения можно найти в разборе примера из теории. Здесь же мы просто применим полученные знания.

В город KKK можно напрямую попасть только из городов GGG, EEE и DDD. Обозначим за SKS_KSK​ количество способов попасть в город KKK. Тогда:

SK=SG+SE+SDS_K = S_G + S_E + S_DSK​=SG​+SE​+SD​

Рассмотрим каждое слагаемое отдельно. Начнем с SGS_GSG​:

SG=SF+SI+SHSG=SI⏟SF+SI+SC⏟SHSG=SI+SI+SI+SB⏟SCSG=3SI+SB S_G = S_F + S_I + S_H \\ S_G = \underbrace{S_I}_{S_F} + S_I + \underbrace{S_C}_{S_H} \\ S_G = S_I + S_I + \underbrace{S_I + S_B}_{S_C} \\ S_G = 3 S_I + S_BSG​=SF​+SI​+SH​SG​=SF​SI​​​+SI​+SH​SC​​​SG​=SI​+SI​+SC​SI​+SB​​​SG​=3SI​+SB​

Из AAA попасть в III можно только двумя способами, так что SI=2S_I = 2SI​=2, а в BBB только одним — SB=1S_B = 1SB​=1:

SG=3⋅2+1=7S_G = 3 \cdot 2 + 1 = 7SG​=3⋅2+1=7

Остальные слагаемые (SES_ESE​ и SDS_DSD​) точно так же сводим к SIS_ISI​ и SBS_BSB​:

SE=SF=SI=2SD=SJ+SH=⋯=2SI+SB=2⋅2+1=5 S_E = S_F = S_I = 2 \\ S_D = S_J + S_H = \cdots = 2 S_I + S_B = 2 \cdot 2 + 1 = 5SE​=SF​=SI​=2SD​=SJ​+SH​=⋯=2SI​+SB​=2⋅2+1=5

Наконец можем получить финальный ответ:

SK=SG+SE+SD=7+2+5=14S_K = S_G + S_E + S_D = 7 + 2 + 5 = 14SK​=SG​+SE​+SD​=7+2+5=14
Ответ

141414

Правило сложения

Сколько существует способов выбрать один объект из нескольких групп объектов? На этот вопрос отвечает правило сложения, одно из основных комбинаторных правил.
Во что поиграть?

Петя хочет поиграть в компьютерную игру. В его библиотеке игр есть 101010 стрелялок, 555 стратегий и 333 квеста. Сколькими способами Петя может выбрать одну игру?

Решение

По правилу суммы у Пети 10+5+3=1810 + 5 + 3 = 1810+5+3=18 способов выбрать одну игру.

Ответ

181818

Да будет свет!

В темной комнате есть 333 настольные лампы, 222 лампочки на потолке, связанные с выключателем около двери, камин и 333 зашторенных окна. Сколько существует вариантов осветить эту комнату?

Решение

Для освещения комнаты можно: включить одну из 333 настольных ламп, включить лампочки на потолке с помощью выключателя, разжечь камин, снять шторы с одного из 333 окон. По правилу сложения получается 3+1+1+3=83 + 1 + 1 + 3 = 83+1+1+3=8 способов осветить комнату.

Ответ

888

Чай, кофе, потанцуем?

На дискотеке есть 444 парня и 333 девушки. Артем станцевал 101010 парных танцев, Василий 333, Петр и Игорь по 444. Света станцевала 222 парных танца, Аня 888. Сколько парных танцев станцевала Ксюша?

Парни танцуют только с девушками и наоброт.

Указание

Танцы каждого парня образуют группу танцев. Эти группы танцев не пересекаются.

Решение

Начнем с парней. Имеем 444 группы танцев, по одной группе на каждого из парней. Группы эти не пересекаются, так как к каждой группе танцев привязан свой парень. По правилу суммы получаем 10+3+4+4=2110 + 3 + 4 + 4 = 2110+3+4+4=21 парный танец.

В парных танцах парни танцуют с девушками. Вся описаная логика с группами и правилом суммы работает и на них тоже. Раз парни станцевали 212121 танец, то и девшуки в сумме тоже станцевали 212121 танец.

2⏟Света+8⏟Аня+x⏟Ксюша=21\underbrace{2}_{\substack{\text{Света}}} + \underbrace{8}_{\substack{\text{Аня}}} + \underbrace{x}_{\substack{\text{Ксюша}}} = 21Света​2​​+Аня​8​​+Ксюша​x​​=21

Решив простейшее уравнение получаем, что Ксюша станцевала 111111 парных танцев.

Ответ

111111

Невероятные восьмерки

Сколько трехзначных чисел в своей записи содержат ровно две восьмерки?

Указание

Разбейте все трехзначные числа с двумя восьмерками на классы. Посчитайте, сколько чисел в каждом классе и используйте правило сложения.

Решение

Разобьем все возможные трехзначные числа ровно с двумя восьмерками на 333 класса: класс 88x88x88x, класс 8x88x88x8 и класс x88x88x88. В классах 88x88x88x и 8x88x88x8 вместо xxx могут быть любые цифры, кроме 888, то есть 0,1,2,3,4,5,6,7,90, 1, 2, 3, 4, 5, 6, 7, 90,1,2,3,4,5,6,7,9 (всего 999 вариантов). Получается, в этих двух классах у нас по 999 чисел. В классе x88x88x88 всего 888 чисел, потому что 000 не может быть в начале трехзначного числа.

По правилу сложения имеем 262626 трехзначных чисел, в записи которых есть только две восьмерки:

9⏟88x+9⏟8x8+8⏟x88=26\underbrace{9}_{\substack{\\ \normalsize 88x}} + \underbrace{9}_{\substack{\\ \normalsize 8x8}} + \underbrace{8}_{\substack{\\ \normalsize x88}} = 2688x​9​​+8x8​9​​+x88​8​​=26
Ответ

262626

Лишние отличники

Экзамен по математике на «555» сдали 999 учеников. Экзамен по русскому языку на «555» сдали 777 учеников. 333 ученика получили отлично на обоих экзаменах. Сколько человек имеет пятерку по математике или по русскому языку?

Указание

Перед применением правила суммы не забудьте проверить, не пересекается ли группа отличников по математике с группой отличников по русскому языку.

Решение

У нас есть две группы учеников: отличники по математике и отличники по русскому языку. В первой группе 999 человек, во второй 777. Но эти группы пересекаются, потому что и там и там есть одни и те же 333 ученика, которые получили «555» по обоим предметам.

Если мы просто сложим 999 и 777, то с добавлением 777 мы повторно учтем 333-х отличников, которые и так есть среди 999 человек первой группы. По правилу суммы получается всего 9+7−3=139 + 7 - 3 = 139+7−3=13 отличников.

Ответ

131313

Меньше знаешь, крепче спишь

В группе 404040 человек. Из них 202020 знают комбинаторику, 303030 знакомы с теорией вероятностей. 151515 человек обе темы. Сколько человек не знает ни комбинаторику, ни теорию вероятностей?

Указание

Задача решается так же, как и «Лишние отличники», только в конце надо выполнить еще одно действие.

Решение

В задаче есть два пересекающихся класса. Поэтому мы можем использовать правило суммы и не забыть вычесть количество повторящихся объектов.

20+30−15=3520 + 30 - 15 = 3520+30−15=35

Значит, «комбинаторику или теорию вероятности» знают 353535 человек. Но нас просят найти тех, кто ничего не знает. Для этого вычитаем 353535 человек, которые хоть что-то знают из общего количества человек: 40−35=540 - 35 = 540−35=5.

555 человек не знают ни комбинаторику, ни теорию вероятности.

Ответ

555

Работа не волк, в лес не убежит

Местная газета опросила 200200200 молодых людей. 170170170 из них учатся в университете. 454545 человек работают. 272727 не работают и не учатся.

  • Сколько молодых людей учатся и работают?

  • Сколько человек только учатся?

  • Сколько человек только работают?

Указание

Задача является развитием идей задач «Меньше знаешь, крепче спишь» и «Лишние отличники». Во всех трех задачах по сути выполняются одни и те же вычисления, просто с разными неизвестными.

Сначала найдите, сколько человек занимаются хоть чем-то. Потом используйте правило суммы.

Решение

Сначала определим, сколько человек заняты хоть чем-то: работой, учебой или и тем и другим сразу. Для этого из общего количества человек вычтем тех, кто ничем не занят:

200−27=173200 - 27 = 173200−27=173

Учащиеся и работащие образуют два класса. По правилу суммы получается 170+45=215170 + 45 = 215170+45=215 человек. Но нам уже точно известно, что чем-то занимаются только 173173173 человека. Значит наши классы пересекаются и есть 215−173=42215 - 173 = 42215−173=42 человека, которые одновременно работают и учатся.

Итак, 424242 человека учатся и работают. Тогда 170−42=128170 - 42 = 128170−42=128 человек только учатся, а 45−42=345 - 42 = 345−42=3 человека только работают.

Ответ

424242 человека учатся и работают. 128128128 человек только учатся. 333 человека только работают.

Решений мало не бывает

Сколько положительных целых решений имеет следующее выражение?

−7<x≤5  или  4<x<100-7 < x \leq 5 \ \text{ или } \ 4 < x < 100−7<x≤5  или  4<x<100
Указание

Положительные целые числа это числа вида 1,2,3,…1, 2, 3, \ldots1,2,3,….

Обращайте внимние на строгость знаков неравенства.

Решения первого неравества это первый класс, второго — второй. Не забудьте проверить их на пересечение.

Решение

У нас есть два класса, каждый из которых связан со своим неравенством. Решениями первого неравенства являются целые числа от 111 до 555 включительно, то есть 666 чисел. У второго неравенства решения это все целые числа от 555 до 999999, то есть 959595 чисел.

Классы у нас пересекаются. Число 555 одновременно является решением как первого, так и второго неравенства, поэтому по правилу суммы всего имеем 6+95−1=1006 + 95 - 1 = 1006+95−1=100 положительных решений.

Ответ

100100100

SK=SG+SE+SDS_K = S_G + S_E + S_DSK​=SG​+SE​+SD​

Послушный робот

Надежда создала робота, который умеет делать три арифметических действия:

  1. Прибавить 111

  2. Прибавить 333

  3. Умножить на 222

Сколько различных цепочек из номеров действий можно дать роботу, чтобы он из числа 555 получил число 171717?

Указание

Принцип решения почти такой же, как и рассмотренного в теории примера.

Решение

Общие рассуждения и пояснения насчет хода решения можно найти в разборе похоже примера из теории. Здесь же мы просто применим полученные знания.

Обозначим за RnR_nRn​ количество цепочек действий, которые дают в результате число nnn. Если число nnn нечетное, то получить его можно только из числа n−1n-1n−1 с помощью действия (1)(1)(1) или из числа n−3n-3n−3 с помощью действия (2)(2)(2). Действие (3)(3)(3) не поможет, так как любое число при умножении на 222 становится четным, а мы рассматриваем нечетное nnn.

Rn=Rn−1+Rn−3(n нечетное)R_n = R_{n-1} + R_{n-3} \quad (n \ \text{нечетное})Rn​=Rn−1​+Rn−3​(n нечетное)

Rn=Rn−1+Rn−3+Rn2(n четное)R_n = R_{n-1} + R_{n-3} + R_{\frac{n}{2}} \quad (n \ \text{четное})Rn​=Rn−1​+Rn−3​+R2n​​(n четное)

Дай... пять!?
Сложная

В сказочной стране живут невероятные жители. У них целых 333 руки: левая, средняя и правая! На каждой руке есть как минимум один палец, а в сумме их всего 171717. Сколько комплектов перчаток должно быть в продаже, чтобы каждый житель смог выбрать себе подходящий?

Указание

Разбейте всех комплекты перчаток на классы, зафиксировав минимальное количество пальцев для каждого класса на одной перчатке и перебирая возможные варианты для двух оставшихся.

Например, K1K_1K1​ будет означать класс вариантов комплектов, где у одной перчатки ровно 111 палец, на остальных не меньше 111 пальца, а в сумме на двух оставшихся перчатках должно быть 161616 пальцев. Потом рассмотрите классы K2K_2K2​, K3K_3K3​ и так далее.

Решение

Описать комплект перчаток можно следующим образом: { 1,2,14 }\set{1,2,14}{1,2,14}. Это означает, что в комплекте три перчатки на 111, 222 и 141414 пальцев, в сумме 171717. Причем совершенно не имеет значения порядок этих чисел: комплекты { 1,2,14 }\set{1,2,14}{1,2,14} и { 14,1,2 }\set{14,1,2}{14,1,2} одинаковые.

Теперь разобьем все варианты комплектов на классы. В класс K1K_1K1​ поместим варианты комплектов, в которых есть рука с 111 пальцем и меньше 111 пальца на руках быть не может. Тогда в сумме на двух оставшихся руках должно быть 161616 пальцев. Математически это можно описать так:

{1,x,y} причем x,y≥1, x+y=16\set{1, x, y} \ \text{причем} \ x,y \geq 1, \ x + y = 16{1,x,y} причем x,y≥1, x+y=16

Все возможные варианты можно перебрать вручную, их тут мало. Получаем 888 комплектов перчаток:

{1,1,15}, {1,2,14}, …,{1,7,9}, {1,8,8}\set{1, 1, 15}, \ \set{1, 2, 14}, \ \ldots, \set{1,7, 9}, \ \set{1, 8, 8}{1,1,15}, {1,2,14}, …,{1,7,9}, {1,8,8}

Напоминаю, что комплект { 1,7,9 }\set{1, 7, 9}{1,7,9} это точно такой же комплект, что и { 1,9,7 }\set{1, 9, 7}{1,9,7}, поэтому последний мы не считаем. Идем только по возрастанию количества пальцев на второй руке и уменьшению на третьей. Так мы точно не пропустим ни один комплект в классе.

Теперь рассмотрим класс K2K_2K2​ в котором на одной руке есть 222 пальца и меньше 222 пальцев на других руках нет. Если мы допустим, что на какой-то руке будет меньше 222 пальцев (то есть 111 палец), то мы повторно учтем комплекты перчаток, которые уже есть в классе K1K_1K1​. В сумме на двух оставшихся руках должно быть 151515 пальцев:

{1,x,y} причем x,y≥2, x+y=15\set{1, x, y} \ \text{причем} \ x,y \geq 2, \ x + y = 15{1,x,y} причем x,y≥2, x+y=15

Получаем 555 комплектов перчаток:

{2,2,13}, {2,3,12}, …, {2,7,8}\set{2, 2, 13}, \ \set{2, 3, 12}, \ \ldots, \ \set{2, 7, 8}{2,2,13}, {2,3,12}, …, {2,7,8}

Аналогичным образом строим класс K3K_3K3​ и находим еще 555 возможных вариантов комплектов перчаток:

{3,3,11}, {3,4,10}, …,{3,7,7}\set{3, 3, 11}, \ \set{3, 4, 10}, \ \ldots, \set{3, 7, 7}{3,3,11}, {3,4,10}, …,{3,7,7}

В классе P4P_4P4​ будет уже 333 вида комплектов, в классе P5P_5P5​ — 222. А в классе P6P_6P6​ и дальше никаких комплектов нет. Ведь уже для P6P_6P6​ в сумме на две оставшиеся руки нужно получить 111111 пальцев. Но минимальная сумма, которая дает 111111 это 6+56 + 56+5, а перчатки с 555 пальцами в классе P6P_6P6​ быть не может.

Обратите внимание, что классы P1,…,P5P_1, \ldots, P_5P1​,…,P5​ не пересекаются. В P1P_1P1​ у всех комплектов есть перчатка с одним пальцем и в то же время во всех остальных классах перчаток с одним пальцем нет. С другими классами точно так же. Раз классы не пересекаются, мы можем спокойно использовать правило суммы:

8⏟P1+5⏟P2+5⏟P3+3⏟P4+2⏟P5=23\underbrace{8}_{\small P_1} + \underbrace{5}_{\small P_2} + \underbrace{5}_{\small P_3} + \underbrace{3}_{\small P_4} + \underbrace{2}_{\small P_5} = 23P1​8​​+P2​5​​+P3​5​​+P4​3​​+P5​2​​=23

Итак, в продаже должно быть 232323 комплекта перчаток.

Ответ

232323

Радиоактивные трехрукие

Задача с трехрукими но теперь после радиоционной катастрофы у них не ровно 17 пальцев, а не больше 15. Решать придется графически через нанесение точек на оси координат.

Чертова лестница
Нестандартная

Лестница состоит из 999 ступенек. Алексей может подниматься ступенька за ступенькой, а может перешагивать через одну. Сколько существует способов подняться по лестнице? А если в лестнице 111111 ступенек?

Указание

Принцип решения очень похож на пример с путешествиями из теории. Только здесь не точки, а ступеньки. Например, попасть на ступень 999 можно только со ступенек 888 или 777.

Решение

Построение модели

Сходу слона съесть не выйдет, поэтому будем есть по кусочкам. В данном случае рассмотрим какую-нибудь одну ступеньку. Возьмем для примера девятую ступень. Попасть на нее можно только с восьмой ступени (поднявшись на одну ступеньку) или с седьмой ступени (переступив через одну). Других вариантов нет.

На ступеньку 9 можно попасть только со ступени 8 и ступени 7.

Представим теперь, что на восьмую ступеньку можно попасть 555 способами. Значит и на девятую ступеньку с восьмой можно попасть тоже только 555-ю способами — совершив один шаг.

Теперь представим, что на седьмую ступеньку можно попасть 777 способами. Тогда, повторяя рассуждения как с восьмой ступенькой, с седьмой на девятую ступень тоже можно попасть только 777 способами.

Получается, способов попасть на девятую ступеньку столько же, как и способов попасть на «восьмую или седьмую» ступеньки. Причем классы способов попасть на ступеньку 888 и на ступеньку 777 не пересекаются! Одинаковых путей нет, ведь любой путь из класса «Ступень 888» содержит в себе восьмую ступеньку, которой нет ни в одном пути из класса «Ступень 777».

Итак, у нас есть два непересекающихся класса, а значит мы можем использовать правило суммы: 5+7=135 + 7 = 135+7=13 способов попасть на девятую ступеньку.

Математическая модель

Фишка в том, что подобные рассуждения можно провести для любой ступеньки. Тогда количество способов попасть на ступеньку nnn равно сумме количества способов попасть на ступеньки n−1n-1n−1 и n−2n-2n−2. Если обозначить количество способов попасть на ступень nnn за SnS_nSn​, то получаем формулу:

Sn=Sn−1+Sn−2S_n = S_{n-1} + S_{n-2}Sn​=Sn−1​+Sn−2​

Решение задачи

Попасть на первую ступеньку можно только одним способом S1=1S_1 = 1S1​=1. На вторую уже есть два способа S2=2S_2 = 2S2​=2: с первой ступеньки и сразу с пола переступив через первую. Для третьей ступеньки уже можно воспользоваться выведенной формулой:

S3=S2+S1=2+1=3S_3 = S_2 + S_1 = 2 + 1 = 3S3​=S2​+S1​=2+1=3

Ну и продолжаем дальше считать с помощью нашей формулы до победного:

S1S_1S1​S2S_2S2​S3S_3S3​S4S_4S4​S5S_5S5​S6S_6S6​S7S_7S7​S8S_8S8​S9\textcolor{#a93838}{S_9}S9​S10S_{10}S10​S11\textcolor{#a93838}{S_{11}}S11​
11122233355588813131321212134343455\textcolor{#a93838}{55}55898989144\textcolor{#a93838}{144}144

Интересная взаимосвязь

Обратите внимение на полученную последовательность чисел:

1,2,3,5,8,12,…1, 2, 3, 5, 8, 12, \ldots1,2,3,5,8,12,…

Каждый следующий элемент этой последовательности получается через сложение двух предыдущих элементов. Это широко известные «Числа Фибоначчи». Они всплывают в самых разных областях, причем не только в математике, но даже и в природе. Только что мы выяснили, что они связаны и с подъемом на самую обыкновенную лестницу.

Удивительно...

Ответ

На лестницу из 999 ступенек можно подняться 555555 разными способами. А на лестницу из 111111 ступенек — 144144144.

Задачи с шахматной доской

Дана шахматная клетка (координаты). Сколько существует клеток, из которых конь может попасть в данную клетку за два хода? Можно усложнить условие, убрав строгое ограничение на количичество ходов: не «за два хода», а «не более чем за три хода». Тогда могут появиться интересные циклы: конь может отойти назад от конечной клетки, а потом вернуться обратно. Интересный факт, конь может начинать прямо из конечной клетки! Это могут забыть посчитать!

Даны фигура в начальной клетке и конечная клетка. Фигура может походить вверх, вправо и по диагонали. Причем она не может дважды походить одинаково. Сколькими способами фигура может попасть в конечную клетку?

Задачи на пошаговый бой

Можно попробовать развить подход с тем, что некоторые действия можно использовать не всегда. Например, есть монстр с 505050 единицами здоровья. А герой может совершать разные типы атаки, которые отнимают разное количество здоровья босса. Но особые атаки имеют «время отката» и после их использования они недоступны какое-то количество ходов. Найти, сколько есть способов убить монстра.
